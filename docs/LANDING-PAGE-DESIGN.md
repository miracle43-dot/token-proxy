## 一、设计方向概述

### 1.1 整体定位

- **关键词**：专业、科技感、开发者导向、高转化
- **视觉基调**：深色背景 + 品牌紫点缀，参考 evolai.cn 的简洁克制风格，避免过度装饰
- **用户画像**：开发者、技术爱好者、企业技术负责人
- **核心目标**：引导用户完成注册 → 充值 → 调用 API 的最短路径

### 1.2 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色（Brand Purple） | `#7C3AED` | CTA按钮、强调元素、Logo |
| 主色亮调（Hover） | `#8B5CF6` | 按钮悬停态、链接 |
| 主色暗调（Pressed） | `#6D28D9` | 按钮按下态 |
| 深色背景 | `#0F0F1A` | 全局默认背景 |
| 卡片背景 | `#1A1A2E` | 区块背景、卡片 |
| 卡片边框 | `#2D2D44` | 卡片描边、分隔线 |
| 文字主色 | `#F5F5FA` | 主标题、正文 |
| 文字次色 | `#A0A0B8` | 副标题、说明文字 |
| 文字弱色 | `#6B6B80` | 占位符、禁用态 |
| 成功色 | `#10B981` | 状态徽章、正向指标 |
| 警告色 | `#F59E0B` | 用量警告等 |
| 错误色 | `#EF4444` | 错误提示 |

### 1.3 字体系统

```
字体族：
- 标题： Inter（英文字体），"PingFang SC" / "Microsoft YaHei"（中文备选）
- 正文： Inter，"PingFang SC" / "Microsoft YaHei"
- 代码/数字：JetBrains Mono（等宽，用于价格、API用量等）

字号规范：
- Hero 标题：64px / 72px（桌面），40px / 48px（移动）| font-weight: 800
- Section 标题：40px / 48px（桌面），28px / 36px（移动）| font-weight: 700
- 卡片标题：24px / 32px | font-weight: 600
- 正文：16px / 24px | font-weight: 400
- 辅助文字：14px / 20px | font-weight: 400
- 最小字号：12px / 16px
```

### 1.4 间距系统

基于 8px 网格：
- `xs`: 4px
- `sm`: 8px
- `md`: 16px
- `lg`: 24px
- `xl`: 32px
- `2xl`: 48px
- `3xl`: 64px
- `4xl`: 96px
- Section 间距：`96px`（桌面），`64px`（移动）

### 1.5 圆角与阴影

```
卡片圆角：16px
按钮圆角：8px（实体按钮）/ 6px（次要按钮）
输入框圆角：8px
图片/截图圆角：12px

阴影（仅用于卡片浮起态）：
box-shadow: 0 4px 24px rgba(124, 58, 237, 0.15), 0 1px 4px rgba(0,0,0,0.4)
```

### 1.6 动效规范

```
原则：克制、快速、不打断阅读
- 过渡时长：200ms（交互反馈）/ 300ms（区块进入）
- 缓动函数：cubic-bezier(0.4, 0, 0.2, 1)
- 滚动进入：fade-in + translateY(20px → 0)，用于各 Section
- 按钮悬停：背景色变亮 + scale(1.02)
- 卡片悬停：边框颜色变为主色，box-shadow 浮现
- 禁止：页面自动播放动画、闪烁元素
```

### 🔵 Section 5 — Quick Start（快速开始）

**标题**：`3 步完成接入`

**布局**：横向3步（桌面），竖向时间线（移动）

**时间线样式**：

```
  ① ──────────→  ② ──────────→  ③
  注册账号        充值余额       开始调用
  Register       Top Up        Start Calling
```

**每步内容**：

| 步骤 | 图标 | 标题 | 描述 |
|------|------|------|------|
| 01 | 👤 | 注册账号 | 邮箱注册，30秒完成，无需手机号 |
| 02 | 💳 | 充值余额 | 支付宝/微信/银行卡，按量计费 |
| 03 | 🚀 | 开始调用 | 替换 API Base URL，立即使用 |

**代码片段示例**（展示在第3步旁边）**：

```python
# Python 示例
import openai

client = openai.OpenAI(
    api_key="tkp_xxxxxxxxxxxx",      # ← 替换为你的 Key
    base_url="https://api.tokenproxy.com/v1"  # ← TokenProxy 专用端点
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)
```

**规范**：
- 步骤连接线：`1px dashed #2D2D44`，带箭头
- 步骤编号圆圈：直径 48px，背景 `#7C3AED`，数字白色
- 代码区：背景 `#12121F`，圆角 12px，JetBrains Mono 字体，14px
