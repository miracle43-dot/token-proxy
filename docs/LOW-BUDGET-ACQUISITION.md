# 低预算快速获取开发者客户方案

## 一、最快获客平台优先级

| 平台 | 获客速度 | 成本 | 难度 |
|------|----------|------|------|
| 掘金 | ★★★★★ | 免费 | 低 |
| GitHub | ★★★★☆ | 免费 | 中 |
| V2EX | ★★★★☆ | 免费 | 低 |
| 技术社群（微信/飞书/Discord） | ★★★★☆ | 免费 | 中 |
| 开源中国 | ★★★☆☆ | 免费 | 低 |
| 知乎/思否 | ★★★☆☆ | 免费 | 中 |

---

## 二、每种渠道具体动作

### 1. 掘金（优先级：最高）
- **发技术干货文章**，标题参考：
  - 《我用 Claude API 做了个 XX，比官方便宜 70%》
  - 《手把手教你白嫖各大 AI API（附代码）》
  - 《API 代理避坑指南：延迟/稳定性/成本全测评》
- 每周发 2 篇，文章末尾软植入官网链接
- 积极回复评论，主动在相关话题下发帖

### 2. GitHub（优先级：高）
- 创建开源工具/脚本仓库（如 API 调用封装库、价格对比脚本）
- README 放官网链接
- 在热门 AI 相关 repo 下提 Issue/PR 刷存在感
- 搜索 "OpenAI API" "Claude API" 等关键词，找使用文档，顺手提供更优解

### 3. V2EX（优先级：高）
- 在 [API] [开发者] [AI] 等节点发技术帖
- 内容方向：踩坑经历、性价比对比、工具推荐
- 账号签名档写官网地址

### 4. 技术社群（优先级：高）
- 微信群/飞书群/Discord：找 AI 开发者社群加入
- 潜水观察 2-3 天，然后发有价值的回答/分享
- 每周在群里提供 1-2 次免费技术咨询，建立信任
- 避免硬广，被踢前先加私聊

### 5. 开源中国（优先级：中）
- 同步掘金文章，或发独立技术帖
- 参与问答板块，解答 API 相关问题

### 6. 知乎/思否（优先级：中）
- 回答 "如何调用 GPT API" "API 代理哪个好" 等问题
- 软植入使用体验和官网链接

## 三、第一周立刻做的 3 件事

### Day 1-2：发第一篇掘金文章
- 选题：**《国内调用 Claude/GPT API 最省钱方案》**
- 内容：实测对比 + 代码示例 + 官网引导
- 预计耗时：2-3 小时
- 发完在群里同步

### Day 3-4：入驻 V2EX + 开源中国
- 账号注册 + 签名档设置（官网地址）
- 各发一篇技术帖（可与掘金内容复用）
- 找 2-3 个相关节点发布

### Day 5-7：加入 3-5 个技术社群
- 搜索微信/飞书 "AI 开发者" "GPT API" "Claude" 相关群
- 申请加入，每天花 30 分钟观察和互动
- 准备一个自我介绍：「在做 AI API 聚合平台，欢迎交流」

---

## 四、核心原则

1. **先给价值，再提产品** — 先让别人觉得你有用，再自然带出产品
2. **内容可复用** — 一篇干货发多个平台，不重复造轮子
3. **数据验证渠道** — 第一周结束后统计各渠道 UV/注册转化，优先放大有效渠道
