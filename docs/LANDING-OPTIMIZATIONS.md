# TokenProxy 落地页优化建议

> 审查时间：2026-04-15
> 审查人：小美（UI/UX 设计师）

---

## 整体评估

当前页面结构清晰（Hero → 社会证明 → 功能 → Playground → 定价 → 快速开始 → FAQ → Footer CTA），代码质量好，响应式设计完整，Mac 窗口 terminal 演示是一大亮点。

主要改进空间在**信任感强化**、**品牌差异化**、**CTA 转化效率**三方面。

---

## 建议 1：Hero CTA 文案从"立即开始"改为具体利益点

**问题**

- 主 CTA "立即开始 →" 过于泛化，没有传达注册即送 $5 体验金的核心利益
- 副标题虽然写了"无需信用卡，注册即送 $5 体验金"，但作为次要信息容易被忽略

**优化方案**

将 Hero 区域的两个按钮改为：

```
[主 CTA]  →  免费注册，送 $5 体验金 →
[次 CTA]  →  查看文档
```

"免费注册，送 $5 体验金" 直接回答了用户两个问题：**要钱吗？**（免费）和**能干什么？**（$5 体验金）。比"立即开始"转化率更高。

---

## 建议 2：社会证明数据加锚点，强化信任感

**问题**

当前 proof-bar 数据：

| 数据 | 问题 |
|------|------|
| 12,000+ 已服务开发者 | "已服务"语义模糊，数字没有来源背书 |
| 1.2 亿 API 调用 | 缺少时间维度，是累计还是每月？ |
| 180 天 稳定运行 | 静态数字，无法体现当下状态；180天≈6个月，略显单薄 |
| 15+ 支持模型 | 数字本身可用，但孤立展示说服力弱 |

**优化方案**

将静态数据替换为**动态状态展示 + 来源背书**：

```vue
<!-- 替换 proofItems -->
{ number: '99.9%', label: 'SLA 可用率' },
{ number: '12,000+', label: '活跃开发者' },
{ number: '1.2 亿/月', label: 'API 调用（本月）' },
{ number: '15+', label: '支持模型' },
```

关键变化：
- "180 天" → **"99.9% SLA"**，用 SLA 百分比比"180天"更有说服力
- "已服务开发者" → **"活跃开发者"**，明确时间范围
- "1.2 亿" → **"1.2 亿/月"**，消除歧义
- 如果能接入真实 uptime 监控数据，可加一个 live status badge（🟢 正常运行）

## 建议 3：Playground 区域加引导文案，降低理解门槛

**问题**

Playground 是全页最有差异化的模块（右上角 Mac 窗口直观展示 API 调用流程），但当前标题：

```
Playground
在线调试，所见即所得
```

**"所见即所得"** 是陈词滥调，用户看完还是不知道这个 Playground 能做什么。右侧 Playground 界面本身内容复杂（API Key 列表、Usage、Model 选择、Request/Response），缺乏上下文会让非专业用户产生困惑。

**优化方案**

在 Playground Mac 窗口**上方**或**右侧**增加一句明确的行为引导文案：

```
左侧选择模型，右侧实时查看响应结果和费用 — 无需写代码
```

同时优化 playground 标题为更具体的功能描述：

```
在线调试
选择一个模型，输入问题，立刻看到 API 响应和费用
```

## 建议 4：品牌差异化 — LOGO 点改为有辨识度的图形符号

**问题**

当前 Logo：
```
● TokenProxy
```
Logo 点是一个纯色圆点 `●`，没有任何品牌辨识度。TokenProxy 的竞品（OpenAI、Anthropic 等）都有独特的图形符号。

**优化方案**

将 Logo 符号改为带品牌色的抽象图形，例如：

```vue
<!-- 方案 A：箭头/连接符号 -->
<span class="logo-icon">
  <svg width="24" height="24" viewBox="0 0 24 24" fill="none">
    <circle cx="8" cy="12" r="4" fill="#7C3AED"/>
    <circle cx="16" cy="12" r="4" fill="#10B981"/>
    <path d="M12 12h4" stroke="#7C3AED" stroke-width="2"/>
  </svg>
</span>
```

两个连接的点 = proxy 的视觉隐喻，紫色（API）+ 绿色（连接成功）。

## 建议 5：定价表加"推荐"标识，引导用户决策

**问题**

当前定价页是纯表格，GPT-4o / GPT-4o-mini / Claude 3.5 Sonnet / Claude 3.5 Haiku / Gemini 1.5 Pro 五行并列，没有优先级区分。用户面对多个选项容易产生决策瘫痪。

**优化方案**

在价格最高的 GPT-4o 行（或性价比最高的 GPT-4o-mini 行）加一个视觉标识：

```
[推荐] GPT-4o-mini   $0.15 / M   $0.60 / M   [官方 8 折]  ✅ 最受欢迎
```

具体改动：
- 在最性价比的方案（GPT-4o-mini）上加 `[推荐]` 标签（紫色边框+背景）
- 在用户量最大的方案旁加 `✅ 最受欢迎`
- 价格数字字体加粗，与备注文字拉开层级
